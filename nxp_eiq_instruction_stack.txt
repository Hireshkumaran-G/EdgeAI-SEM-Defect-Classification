NXP eIQ Deployment Instruction Stack (Conceptual)

1. Input preprocessing
   - Grayscale SEM image
   - Resize to 128x128
   - Normalize to [0,1]
   - Replicate to 3 channels

2. Inference engine
   - MobileNetV3-Small
   - ONNX format
   - INT8 quantized model
   - Static input shape

3. Execution backend
   - NXP eIQ runtime
   - INT8 arithmetic
   - CPU / NPU acceleration depending on target platform

4. Output postprocessing
   - Softmax
   - Argmax class selection
   - Defect class output

Note:
Final instruction stack generation and performance profiling
will be performed on target NXP edge hardware using eIQ tools.
